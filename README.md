# SPACECRAFT
a simple space invader like game on terminal using `curses` lib.

## BUILD
- Configure 
```sh
meson configure build
```
- Build
```sh
ninja -C build
```

## GAMEPLAY

| Key | Function |
|-----|----------|
| W   | go up    |
| a   | go left  |
| s   | go down  |
| d   | go right |
| q   | quit     |
